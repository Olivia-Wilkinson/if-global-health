# Fractional Dosing Review: Data, Analysis, and Interactive Figures

This repository contains the data extraction materials, analysis files, and figures used for our review of fractional vaccine dosing.

The repository is intended primarily to support **transparency, legibility, and reproducibility**. It is not a software package or a polished analytical toolkit; rather, it provides the working materials behind the review and hosts interactive versions of selected figures.

## What this repository is for

This repo supports the review by making it possible to:

- inspect the extracted dataset used in the project
- see how summary counts and classifications were produced
- view the code and notebooks used to generate the figures
- access interactive Plotly versions of selected figures

## What is included

This repository contains some or all of the following:

- the data extraction sheet used in the review
- analysis notebooks and/or scripts used to summarise the dataset
- exported figures used in the write-up
- interactive Plotly figures hosted via GitHub Pages

## How to use this repository

If you are reading the paper and want to understand or verify the analysis, the most useful places to start are:

1. the extracted dataset
2. the analysis notebook(s) / script(s)
3. the figures folder
4. the interactive figure site

This repository is mainly designed for inspection rather than formal reuse. The analysis is relatively simple and is intended to make the workflow behind the paper transparent!

## Interactive figures

Interactive versions of selected figures are available via GitHub Pages: **https://olivia-wilkinson.github.io/if-global-health**

These are supplementary to the static figures included in the write-up.

## Relationship to the paper

This repository accompanies our review of fractional vaccine dosing. It contains the materials used to support the descriptive analyses, gap analysis, and visualisations reported in the paper.

The core argument of the paper is that fractional dosing should be treated as a **vaccine-specific strategy**, not a generic response to vaccine scarcity. The materials here are intended to make the evidence base and analytical workflow behind that conclusion easier to inspect.

## Reproducibility note

The analyses in this repository were created to support the review write-up. They are included in the interest of openness and transparency, but the repository should be understood as a project archive rather than a fully packaged reproducible pipeline.

If you reference this repository, please cite the associated paper / report as the primary source.
